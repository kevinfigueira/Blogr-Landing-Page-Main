### Visão Geral

#### O desafio

O usuário pode:

- Usufruir do layout do aplicativo independente do tamanho da tela (Design Responsivo);
- Ver os estados ativos e com foco dos elementos;
- Ver cada página e poder navegar pelos menus internos.
